# Ark Recognizer

A recognizer to recognize items of *Arknights* screenshots.

## Principle

1. 对场景图像和模板图像进行 HoughCircles，获取场景与模板图像大小之比。
2. 将场景图像缩放到模板图像大小，并将场景图像和模板图像共同缩放一个固定的 factor。
3. 进行模板匹配。
